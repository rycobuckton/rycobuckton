Medidas relativas
em, ex, rem, vw, vh, %
16px geralmente é igual a 1em
1em geralmente é = a 12pt (segundo meus testes)
logo 1.25em = 15pt
preferir o "em" Exemplo font-size 1.5em;

EM é uma unidade de medida tipográfica. Seu nome está relacionada com a letra “M”, onde o tamanho base dessa unidade deriva da largura da letra M em maiúscula. Dizem que 1em equivale aproximadamente 16 pontos.

Não confundir: (weight = peso) - (width = largura) - (height = altura)
